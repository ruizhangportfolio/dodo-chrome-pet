# Dodo Chrome Pet

Dodo Chrome Pet is a launched Chrome extension that turns Dodo, my real Ragdoll cat and YouTube channel subject, into a calm interactive browser companion.

This repository contains the product brief for the project, including the product context, MVP scope, platform decision, trade-offs, privacy boundaries, early launch evidence, validation plan, and future roadmap.

## Current Evidence Status

The July 2026 launch produced directional YouTube feedback and Chrome Web Store activity. The brief reports those signals with explicit evidence boundaries and does not claim proven retention, attributed conversion, or business impact.

## Product Framing

Dodo Chrome Pet should not be framed as only a cute browser pet.

It is a creator-led product experiment that tests whether passive video affinity can become a daily browser companion experience, while balancing audience retention, low-friction distribution, attention-respecting UX, privacy constraints, and MVP launch scope.
